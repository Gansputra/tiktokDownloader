## Cara Menjalankan
1.  Clone repo ini.
2.  Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```
3.  Jalankan aplikasi:
    ```bash
    python app.py
    ```
4.  Buka browser di `http://127.0.0.1:5000`
